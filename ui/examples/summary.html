Variant Selection Guide

Variant When to use Zone 3 required?
Single Metric Card One KPI needs dedicated real estate Optional
Metric Strip Compare 2–4 KPIs in a single header row No — omit for compactness
Detail Grid Metrics need progress context + drill links Yes — all 3 zones required
Status Roll-Up Health of a multi-component system Optional (link to incident log)
Comparative Card Comparison is the primary message Optional
